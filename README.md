# Aspiiro — Official Website

E-commerce website for Aspiiro, a Gen Z clothing brand based in Pune.

## Live Site
[www.aspiiro.com](https://www.aspiiro.com)

## Stack
- **Frontend** — Framer (visual builder)
- **Payments** — Razorpay (UPI, Cards, Net Banking)
- **Domain** — GoDaddy → connected to Framer

## How Payments Work
Each product has a Buy Now button linked to a Razorpay Payment Link.
Customers pay a minimum 20% deposit at checkout to confirm the order.
Remaining amount is collected before or on delivery.

## Adding New Products
1. Log into Framer
2. Go to the Shop page
3. Upload new product image, add name and price
4. Create a new Razorpay Payment Link for that product (20% of price)
5. Paste the link behind the Buy Now button
6. Publish



## Local Development
This project is built on Framer — no local setup needed.
All edits are made directly in the Framer dashboard.
